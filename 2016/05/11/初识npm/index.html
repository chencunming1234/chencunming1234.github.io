初识npm

npm的安装

npm install -g;

-g表示全局安装

npm adduser 创建账号
命令行创建在终端不用添加，直接绑定机器，在官网创建的话需要手动添加账户信息；

第三方模块：

npm install+模块名称（下载第三方模块到本地）

自定义模块发行到npm

npm init （初始化）
（含有：package.son;index.js;README.md（可选））
npm publish （发行上传）

npm unpublish+模块名称@版本号（删除发行）

sudo npm -g hexo （全局下下载安装hero）
