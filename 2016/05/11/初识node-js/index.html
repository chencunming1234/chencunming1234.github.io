初识node.js

下载安装node.js;

进入官网：https://nodejs.org/en/

做开发的时候最好还是用LTS稳定版的，平时学习的时候可随意！

js与node.js的一些区别


1、b/s系统；
浏览器/服务端

chrome浏览器//两个内核 webkit v8

webkit:页面布局，解析样式
v8:JS解析器
硬件层：

node.js的底层基于c

libuv > libev >连接c/c++ > 硬件层

客户端JS（ECMA/DOM/BOM）
核心为ECMA
DOM/BOM是浏览器自己封装的

在node.js里面只有ECMA，没有BOM DOM window

znode.js的API:process os file buffer util queryting
node.js特色：没有兼容性问题;可运行ecma6;


### node.js的速度远比js快的多

##node.js的安装

到官网下载好安装包以后直接点击运行一直到安装结束


在终端里可检查是否有安装好；下面以mac系统为例



node -v；出现node.js的版本号就说明已经安装完成

##nodejs的一些基本操作：
dirname:目录路径 filename:文件路径
原生模块：
直接可引入：var os=require(‘os’);
var querystring=require(‘querystring’)
var fs=require(‘fs’)
var util=require(‘util’)
var path=require(‘path’)
var vm=require(‘vm’)
还有net,http,https等多种原生模块

获得第三方模块：npm install moduleName;
引入第三方模块：var moduleIn=require(moduleName)

输出自定义js模块：moudble.exports=name;
引入第三方模块：var moduleIn=require(__dirname+”/youname.js”)
